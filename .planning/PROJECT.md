# Personal Linktree

## What This Is

A sleek personal link hub page inspired by Apple's minimal aesthetic. Displays a profile header (name, avatar, tagline) and a clean set of buttons linking to GitHub, Amazon Wishlist, CashApp, and PayPal — built as a single static HTML page.

## Core Value

Visitors can instantly find and tap any of Hector's links from one beautiful, fast-loading page.
