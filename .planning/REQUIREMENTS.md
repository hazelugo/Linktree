# Requirements: Personal Linktree

**Defined:** 2026-03-16
**Core Value:** Visitors can instantly find and tap any of Hector's links from one beautiful, fast-loading page.

## v1 Requirements

### Profile

- [ ] **PROF-01**: Page displays avatar image, name, and tagline in a centered header
- [ ] **PROF-02**: Page has a descriptive document title (browser tab)

### Links

- [ ] **LINK-01**: User can click GitHub button and open GitHub profile in a new tab
- [ ] **LINK-02**: User can click Amazon Wishlist button and open wishlist in a new tab
- [ ] **LINK-03**: User can click CashApp button and open CashApp in a new tab
- [ ] **LINK-04**: User can click PayPal button and open PayPal donation in a new tab

### Design

- [ ] **DESG-01**: Dark background with white text and generous whitespace
- [ ] **DESG-02**: All buttons share a consistent pill/rounded style with subtle hover effect
- [ ] **DESG-03**: Page is fully responsive on mobile and desktop
- [ ] **DESG-04**: SF Pro or system font stack for Apple-feel typography

## v2 Requirements

### Enhancements

- **ENH-01**: Animated entrance (fade/slide in on load)
- **ENH-02**: Light/dark mode toggle
- **ENH-03**: Click analytics

## Out of Scope

| Feature | Reason |
|---------|--------|
| Backend/database | Static page only |
| Social feed embeds | Complexity not warranted |
| Mobile app | Web-first |
