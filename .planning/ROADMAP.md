# Roadmap: Personal Linktree

## Overview

A single-phase build. All 10 requirements are tightly coupled into one deliverable: a complete static HTML page with profile header, four working link buttons, and a polished Apple-inspired dark design. There is no logical split — the page either works as a whole or it doesn't. This phase takes it from nothing to shipped.

## Phase Details

### Phase 1: Build and Ship
**Goal**: Visitors can open the page and immediately find and tap any of Hector's links on a beautiful, fast-loading static page
**Depends on**: Nothing (first phase)
**Requirements**: PROF-01, PROF-02, LINK-01, LINK-02, LINK-03, LINK-04, DESG-01, DESG-02, DESG-03, DESG-04
**Success Criteria** (what must be TRUE):
  1. Visitor sees avatar, name, and tagline centered at the top of the page
  2. Visitor can click each of the four buttons (GitHub, Amazon Wishlist, CashApp, PayPal) and land on the correct destination in a new tab
  3. Page renders correctly on a mobile phone screen with no horizontal scrolling or cramped layout
  4. Page background is dark, text is white, buttons share a consistent pill shape with a visible hover effect, and typography uses a system/SF Pro font stack
  5. Browser tab shows a descriptive page title (not a generic "index" or blank)
**Plans**: 2 plans

Plans:
- [x] 01-01-PLAN.md — HTML structure: document skeleton, profile header (avatar, name, tagline), four link buttons with correct hrefs
- [ ] 01-02-PLAN.md — CSS styling: dark theme, typography, pill buttons, hover effects, responsive layout + human verify
