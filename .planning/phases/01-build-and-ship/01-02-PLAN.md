**Reset and base:**

**Body — dark background, white text, system font stack:**
```css
body {
  background-color: #000000;
  color: #ffffff;
  font-family: -apple-system, BlinkMacSystemFont, "SF Pro Display", "Segoe UI",
               Roboto, "Helvetica Neue", Arial, sans-serif;
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  min-height: 100vh;
  display: flex;
  flex-direction: column;
  align-items: center;

  padding: 64px 24px 48px;
}
```

**Container — narrow centered column:**
```css
.container {
  width: 100%;
  max-width: 480px;
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 0;
}
```

**Profile header:**
```css
.profile {
  display: flex;
  flex-direction: column;
  align-items: center;
  gap: 12px;
  margin-bottom: 40px;
  text-align: center;
}
```

**Avatar — circular, with subtle border:**
```css
.avatar {
  width: 96px;
  height: 96px;
  border-radius: 50%;
  object-fit: cover;
  border: 2px solid rgba(255, 255, 255, 0.15);
}
```

**Name:**
```css
.name {
  font-size: 1.5rem;
  font-weight: 600;
  letter-spacing: -0.02em;
  color: #ffffff;
}
```

**Tagline:**
```css
.tagline {
```

**Links nav — vertical stack:**
```css
.links {
  width: 100%;
  display: flex;
  flex-direction: column;
  gap: 14px;
}
```

**Link button — pill shape, consistent for all four buttons:**
```css
.link-button {
  display: block;
  width: 100%;
  padding: 16px 24px;
  background-color: rgba(255, 255, 255, 0.08);
  color: #ffffff;
  text-decoration: none;
  text-align: center;
  font-size: 1rem;
  font-weight: 500;
  letter-spacing: 0.01em;
  border-radius: 980px;
  border: 1px solid rgba(255, 255, 255, 0.12);
  transition: background-color 0.2s ease, border-color 0.2s ease, transform 0.15s ease;
  cursor: pointer;
  -webkit-tap-highlight-color: transparent;
}
```

**Hover effect:**
```css
.link-button:hover {
  background-color: rgba(255, 255, 255, 0.14);
  border-color: rgba(255, 255, 255, 0.25);
  transform: translateY(-1px);
}
```

**Active/pressed state (important for mobile feel):**
```css
.link-button:active {
  background-color: rgba(255, 255, 255, 0.06);
  transform: translateY(0);
}
```

**Responsive — mobile adjustments at 480px and below:**
```css
@media (max-width: 480px) {
  body {
    padding: 48px 20px 40px;
  }

  .name {
    font-size: 1.375rem;
  }

  .link-button {
    padding: 15px 20px;
    font-size: 0.9375rem;
  }
}
```

**Do NOT:**
- Use external fonts (no Google Fonts @import)
- Add JavaScript-driven animations (no keyframes tied to JS)
- Add any class not present in index.html
- Use `!important`
- Include media queries for dark/light mode toggle (that is deferred to v2/ENH-02)
